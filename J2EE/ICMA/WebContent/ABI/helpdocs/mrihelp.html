Patient MRI Dashboard

The MRI dashboard provides integrated access to the patient’s MRI studies. Each study is presented as a tab and consists of observations along one or more MRI view axis. Each view is associated with some metadata. By default the cine and metadata associated with the first view in the ‘Views‘ portal is loaded when a study tab is selected. The cine can be played or queried using the playable video widget at the centre of the dashboard.

Initial View Controls

1. List of MRI studies currently available for the patient

2. The Date and time of the MRI study

3. Selected metadata for view recording

4. Reset control point information

5. Button to move to the end diastole (ED) frame

6. Button to move to the end systole (ES) frame

7. Button to set the end of cardiac cycle

V. Video widget to play cine associated with view, to seek time points of interest and select control points.

S. View organiser, lists all the views that are associated with the study. The current view displayed in V is highlighted with a light shadow. Another view can be loaded by clicking on its image in the view organiser.

I. Controls to select the imaging slices to be displayed in the 3D widget

P. Control to start the model fitting process

Model fitting and strain analysis

Strain analysis requires a model to be fitted to the wall. The fitting process involves two stages

1. Specify endocardial apex and base-plane at end diastole (ED) and end systole (ES) for at least one long axis view plane

2. Specify the location of right ventricular inserts in at least one short axis view

Specification of long axis control points

An MRI study may contain more than one long axis investigation. Select the one that passes through the middle of the left ventricular (LV) chamber. This view can be determined using the 3D rendering in the widget to the right.

Some MR recordings may cover more than one cardiac cycle. Select the cardiac cycle of interest using the time slider and control 7. To specify the end of cardiac cycle, move the time slider to the desired frame and click on control 7. The beginning of the cardiac cycle is determined by the end diastole frame.

The end diastole frame is specified by the earliest time frame at which endocardial landmarks are specified. When the end diastole frame has been specified, the ED control, 5, is enabled. The user can return to this frame to verify/modify the landmarks by clicking on the control.

The baseplane landmarks at end systole frame are required. The end systole frame occurs post the end diastole frame, using the time slider move to the end systole frame and specify the base plane land marks.

Specification of Short axis RV Inserts

Select a short axis imaging plane that passes close to the base of the LV chamber and allows you to clearly view the locations at which the right ventricular wall meets the LV chamber.

Select these two points by clicking on the widget. No other time specific information is required. Note that the video widget does not know if the cine that it is rendering is a short axis on or a long axis slice. Therefore, it provides the same controls for both cines. The user can easily identify the axis and therefore is expected to specify the control points appropriately. The fitting process uses this meta information (number of points) to determine which of the slices are along the short and long axis.

Model fitting

To fit the model, specify the landmarks and provide a model name. Click on ‘Fit Model’ to initiate the process. A progress bar is launched to indicate the fitting progress. Once a model is fit to the data, the model is added to the ‘Models‘ views and a dialog would notify this.
