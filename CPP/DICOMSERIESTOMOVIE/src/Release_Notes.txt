Requirements
============
ITK 4.2 or higher - (www.itk.org)
libarchive - (www.libarchive.org/)
libcurl - (curl.haxx.se/libcurl/)
Boost C++ libraries - (www.boost.org/)

Calls ffmpeg executable, the location is a passed as an argument  
 
Code organization
=================
The graphs folder provides some information on module interactions in the form of an image file and a dot file (viewed using Graphviz compatible viewers)
