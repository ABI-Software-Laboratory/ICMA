Analysis Variable

The user control Analysis Variable defines a variable that will be used to characterize a model and compare it against other models. The strain values of a standard 16 segment model are named S1...S16.

These values can be used to compose the Analysis Variable. Any expression that is a valid MS EXCEL/ Open Office formula using S1...S16 is admitted.

Modifying expressions

To edit or remove an Analysis variable, double click on the row. This will open a dialog where the variable can be edited.

Report names should not contain spaces or special characters, _ could be used
